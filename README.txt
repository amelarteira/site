ATELIÊ A MEL ARTEIRA — SITE (UMA PÁGINA)

Como usar:
1) Abra o arquivo index.html no navegador (duplo clique).
2) Para publicar: envie a pasta inteira para o seu provedor/host (mantendo a estrutura).

Onde trocar imagens:
- Banner: assets/img/banner.png  (proporção 16:9 — ex.: 1920x1080)
- Logo topo: assets/img/logo.png (proporção 1:1 — ex.: 1200x1200, fundo transparente)
- Ícone de aba: assets/img/bee.png (os favicons já foram gerados em assets/favicon)

Modelos (fotos):
- Fotos principais: ideal 1:1 (ex.: 1200x1200)
- Miniaturas: 1:1 (o site recorta automaticamente)
- Editar as descrições/preços dentro do index.html (busque pelos nomes dos modelos)

Cores/Fontes:
- Ajuste no arquivo assets/css/style.css (variáveis em :root)
